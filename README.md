# Recipe Blog

A simple project for people to share their beloved recipes.

## Installation
If you want, you can start a project manually

Install

```bash
cd RecipeBlog/frontend
npm install
```

```bash
cd RecipeBlog/backend/RecipeBlog/RecipeBlog.API/
dotnet restore
```
also add a an appsettings.json with following values(those must be generated by yourself): 
```bash
{
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft.AspNetCore": "Warning"
    }
  },
  "AllowedHosts": "*",
  "ConnectionStrings": {
    "RecipeDB": "The connection string for mysql db"
  },
  "Jwt": {
    "Issuer": "http://localhost:5004",
    "Audience": "http://localhost:5173",
    "Key": "unique secret"
  },
  "DeeplAPIKey": "api key for Deepl api"
}
```

if you want to start the app from docker compose please add .env in root of the project with following values:
```bash
MYSQL_ROOT_PASSWORD=root password for mysql
MYSQL_DATABASE=name of database
MYSQL_USER=username
MYSQL_PASSWORD=password for user created
JWT_ISSUER=http://localhost:5004
JWT_AUDIENCE=http://localhost:5173
JWT_KEY=unique secret
DEEPL_API_KEY=api key for Deepl api
```

## Run Locally

Go to the project directory

```bash
cd RecipeBlog/frontend
```

Start the frontend

```bash
npm run dev
```


Go to the project directory

```bash
cd RecipeBlog/backend/RecipeBlog/RecipeBlog.API/
```

Start the frontend

```bash
dotnet run
```
or

```bash
docker compose up --build -d
```
from the root of the project

## Usage

```
The ports are 5173 for frontend and 5004 for backend.
Also on endpoint localhost:5004/swagger/index.html there is a swagger showing all the endpoints
```
